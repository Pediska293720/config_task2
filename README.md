# Инструмент визуализации графа зависимостей для менеджера пакетов
## Описание
Инструмент для визуализации графа зависимостей для менеджера пакетов. Реализован на Python с поддержкой настраиваемых параметров через командную строку и обработкой ошибок ввода.

## Задание
Разработать инструмент визуализации графа зависимостей для менеджера пакетов. Создать минимальное настраиваемое CLI-приложение.


## Этап 1 - Минимальный прототип с конфигурацией
- Реализация парсера аргументов командной строки
- Поддержка 6 настраиваемых параметров
- Обработка ошибок ввода

## Функциональность
- **Настраиваемая конфигурация** через аргументы запуска
- **Поддерживаемые параметры**:
  - `--package` — имя анализируемого пакета (обязательный)
  - `--url` — URL репозитория или путь к тестовому файлу
  - `--test-mode` — режим тестового репозитория (флаг)
  - `--output-file` — имя файла для сохранения графа
  - `--depth` — максимальная глубина анализа
  - `--filter` — подстрока для фильтрации пакетов
- **Обработка ошибок**:
  - отсутствие обязательных параметров
  - некорректные значения числовых параметров
  - отсутствие значений для параметров

## Запуск

Минимальный запуск (обязательный параметр --package)
```
py main.py --package request
```
Полная конфигурация
```
py main.py --package requests --url home/users/pediska --test_mode --output_file tree.png --depth 3 --filter "cat"
```
